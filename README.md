# 财经日报雷达

面向财经科技日报的本地 Web 看板，支持来源勾选、RSS 抓取、新闻溯源、章节分析和 Markdown 报告导出。

## ✨ 最新功能

### 🎯 多标签页看板（新）
- **今日概览**：关键指标、综合判断、重点事件（总）
- **章节分析**：六大章节深度分析（分）
- **文章明细**：完整数据表格和筛选（明细）
- **日报生成**：报告预览和下载
- **顶部操作栏**：开始运行、停止、立即抓取、AI 分析等按钮

### 📱 跨设备访问（新）
支持在家里电脑、公司电脑、手机等多设备访问，详见 [跨设备访问指南.md](./跨设备访问指南.md)

## 当前恢复版本包含

- 紫色渐变 Web 看板（原版 + 多标签页版）
- 左侧采集范围按媒体性质分组，卡片一行 3 个
- 国家统计局 RSS：
  - 最新发布：https://www.stats.gov.cn/sj/zxfb/rss.xml
  - 数据解读：https://www.stats.gov.cn/sj/sjjd/rss.xml
- 证券时报、36氪快讯等链接采集器框架
- GPT / Claude 分析配置状态展示接口
- 本地 JSON 存储：`data/local_store.json`

## 🚀 快速启动

### 方式1：使用启动脚本（推荐）

双击 `start_server.bat`，然后访问：

- **新看板**（多标签页）：http://127.0.0.1:5000/tabs
- **原看板**：http://127.0.0.1:5000

### 方式2：手动启动

```bash
python -m web.app
```

如果页面仍是旧版本，按 `Ctrl + F5` 强制刷新。

## 📱 跨设备访问

### 本地/局域网访问
```
http://127.0.0.1:5000/tabs          # 本机
http://[你的IP]:5000/tabs           # 同一WiFi的其他设备
```

### 公网访问（任何地方）⭐

1. 双击 `start_server.bat` 启动服务器
2. 双击 `start_cpolar.bat` 启动内网穿透
3. 复制显示的公网地址 + `/tabs` 即可访问

**详细说明请查看：** [跨设备访问指南.md](./跨设备访问指南.md)
