# 文件索引

- `run_web.bat`：双击启动 Web 服务
- `web/standalone_app.py`：标准库备用 Web 服务
- `web/app.py`：Flask Web 服务
- `web/templates/index.html`：看板页面
- `crawlers/link_collectors.py`：RSS/链接采集器
- `crawlers/cn_sources.py`：统一抓取入口
- `processors/`：去重、聚合、信息抽取、AI 配置
- `database/operations.py`：本地 JSON 数据层
- `output/daily_report.py`：日报生成
- `data/local_store.json`：本地数据
