# 跨设备访问指南

本项目支持在家里电脑、公司电脑、手机等多设备访问看板。

## 📱 三种访问方式

### 方式1：本地访问（仅本机）
```
http://127.0.0.1:5000/tabs
```

### 方式2：局域网访问（同一 WiFi）
```
http://[你的电脑IP]:5000/tabs
```
例如：`http://192.168.1.100:5000/tabs`

**如何查看你的 IP：**
- Windows：打开命令提示符，输入 `ipconfig`，查看 IPv4 地址

### 方式3：公网访问（任何地方）⭐ 推荐
使用 **cpolar** 内网穿透，获得一个公网地址，随时随地访问。

---

## 🚀 快速开始

### 第一步：启动 Flask 服务器

双击运行 `start_server.bat`，看到以下提示即成功：
```
服务已启动！可通过以下方式访问：
  本地访问：
  - http://127.0.0.1:5000/tabs  (新看板)
```

**保持这个窗口打开**，不要关闭。

---

### 第二步：配置 cpolar（仅首次需要）

#### 2.1 注册并下载

1. 访问 https://www.cpolar.com/
2. 注册免费账号
3. 下载 Windows 客户端并安装

#### 2.2 获取 authtoken

登录 cpolar 网站后，在 **[验证](https://dashboard.cpolar.com/auth)** 页面复制你的 authtoken

#### 2.3 配置 authtoken

打开命令提示符（cmd），运行：
```bash
cpolar authtoken [你的token]
```

例如：
```bash
cpolar authtoken 1a2b3c4d5e6f7g8h9i0j
```

> ⚠️ 这一步只需要做一次，后续不用再配置

---

### 第三步：启动 cpolar 隧道

双击运行 `start_cpolar.bat`

你会看到类似以下输出：
```
========================================
cpolar 隧道已建立！
复制下方的公网地址，即可在任何设备访问
========================================

Forwarding  https://abc123.cpolar.cn -> http://localhost:5000
```

**复制这个公网地址**（如 `https://abc123.cpolar.cn`），加上 `/tabs` 即可访问：
```
https://abc123.cpolar.cn/tabs
```

在你的手机、公司电脑、家里电脑的浏览器中打开这个地址，就能看到看板了！

**保持这个窗口打开**，不要关闭。

## 📝 使用注意事项

### 免费版限制

- ✅ 每天可用时长：无限制
- ⚠️ 每次启动 URL 会变（重启后需要复制新地址）
- ⚠️ 有流量限制（个人使用够用）

如果需要固定域名，可以升级付费版（约 10 元/月）

### 两个窗口都要保持打开

使用 cpolar 时需要同时打开：
1. `start_server.bat` - Flask 服务器
2. `start_cpolar.bat` - cpolar 隧道

关闭任意一个都会导致无法访问。

### 如何关闭

在命令行窗口按 `Ctrl+C` 可以停止服务。

## 🔧 故障排查

### 问题1：cpolar 提示 "Flask 服务器未启动"

**解决方法：** 先运行 `start_server.bat`，等服务器启动后再运行 `start_cpolar.bat`

### 问题2：访问公网地址显示 "502 Bad Gateway"

**原因：** Flask 服务器已关闭

**解决方法：** 检查 `start_server.bat` 窗口是否还在运行

### 问题3：cpolar 提示未安装

**解决方法：** 
1. 确认已经从 https://www.cpolar.com/ 下载并安装
2. 重新打开命令提示符
3. 运行 `cpolar version` 测试是否安装成功

### 问题4：局域网无法访问

**解决方法：**
1. 确认两台设备在同一 WiFi
2. 检查防火墙是否允许端口 5000
3. 尝试关闭 Windows 防火墙测试

## 🌟 推荐使用场景

| 场景 | 推荐方式 | 说明 |
|------|---------|------|
| 本机调试 | 本地访问 | 最快 |
| 手机预览（同 WiFi） | 局域网访问 | 无需配置 |
| 远程访问 | cpolar 公网访问 | 最灵活 |
| 公司+家里切换 | cpolar 公网访问 | 同一个地址 |

## 💡 高级技巧

### 配置固定子域名（付费功能）

升级付费版后，可以在 cpolar 控制台配置固定子域名，例如：
```
https://my-finance-radar.cpolar.cn/tabs
```

这样每次启动地址都不变，更方便使用。

### 使用 ngrok（国际版）

如果在国外使用，可以选择 ngrok：
1. 访问 https://ngrok.com/
2. 注册并下载
3. 运行：`ngrok http 5000`
